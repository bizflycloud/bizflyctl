# Scheduled Volume Backup Commands

Configure automated volume backups on a schedule.

## Overview

Scheduled volume backups allow you to automatically create snapshots of volumes on a recurring schedule, ensuring regular backups without manual intervention.

## Commands

Use `bizfly scheduled-volume-backup --help` to see available scheduled backup commands.

## Related Documentation

For detailed scheduled backup management documentation, refer to the Bizfly Cloud documentation or use:

```bash
bizfly scheduled-volume-backup --help
```

## Common Use Cases

### Setting Up Daily Backups

Configure automatic daily snapshots of important volumes to ensure data protection.

### Setting Up Weekly Backups

Configure weekly backups for less critical volumes to reduce storage costs while maintaining backup coverage.
