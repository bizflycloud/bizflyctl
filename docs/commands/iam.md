# IAM (Identity and Access Management) Commands

Manage users, roles, and permissions.

## Overview

IAM commands allow you to manage access control, users, roles, and permissions for your Bizfly Cloud account.

## Commands

Use `bizfly iam --help` to see available IAM commands.

## Related Documentation

For detailed IAM management documentation, refer to the Bizfly Cloud IAM service documentation or use:

```bash
bizfly iam --help
```
