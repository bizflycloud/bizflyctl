# Custom Image Management Commands

Create and manage custom OS images from existing servers or volumes.

## Overview

Custom images allow you to create reusable templates from your configured servers, making it easy to deploy identical server configurations.

## Commands

Use `bizfly custom-image --help` to see available custom image commands.

## Related Documentation

For detailed custom image management documentation, refer to the Bizfly Cloud documentation or use:

```bash
bizfly custom-image --help
```

## Common Use Cases

### Creating Custom Image from Server

1. Configure a server with your desired setup
2. Create a custom image from that server
3. Use the custom image to create new servers with the same configuration
