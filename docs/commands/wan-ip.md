# WAN IP Management Commands

Manage WAN (public) IP addresses.

## Overview

WAN IPs are public IP addresses assigned to servers for internet access.

## Commands

Use `bizfly wan-ip --help` to see available WAN IP commands.

## Related Documentation

For detailed WAN IP management documentation, refer to the Bizfly Cloud documentation or use:

```bash
bizfly wan-ip --help
```
